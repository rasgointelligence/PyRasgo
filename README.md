# PyRasgo
PyRasgo is a free python package for interacting with the Rasgo API and your own data warehouse.

PyRasgo will help you:

- create and manage your metrics
- integrate with Git and dbt
- consume metrics and datasets into pandas dataframes for ad-hoc analysis or modeling
